🤖 AI Chatbot Extension - 테스트 페이지

확장 프로그램 설치 확인
이 페이지는 Chrome Extension AI Chatbot이 제대로 설치되고 작동하는지 확인하는 테스트 페이지입니다.

📋 설치 체크리스트

1. 확장 프로그램 로드 상태 확인 중...

🔧 설치 방법

1. Chrome에서 chrome://extensions/ 접속
2. 우상단 "개발자 모드" 활성화
3. "압축해제된 확장 프로그램을 로드합니다" 클릭
4. 이 프로젝트 폴더 선택
5. 확장 프로그램 아이콘이 툴바에 나타나는지 확인

🚀 기능 테스트

🦙 Ollama 설정 (로컬 AI)

# Ollama 설치 (macOS)
brew install ollama

# Ollama 서비스 시작
ollama serve

# 모델 다운로드 (예: Llama 3)
ollama pull llama3

# 확장 프로그램에서 설정:
- Backend: "Ollama (Local)" 선택
- URL: http://localhost:11434
- 연결 테스트 실행

🔗 n8n 설정 (Webhook)

# n8n 워크플로우 설정:
1. Webhook 노드 추가 (POST 방식)
2. AI 서비스 연결 (OpenAI, Claude 등)
3. Response 노드로 JSON 응답 설정

# 확장 프로그램에서 설정:
- Backend: "n8n Webhook" 선택
- URL: https://your-n8n.com/webhook/chatbot
- 연결 테스트 실행
