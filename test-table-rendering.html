AI 챗봇 테이블 렌더링 및 사이드바 리사이즈 테스트

테스트 1: 마크다운 테이블 렌더링

AI 챗봇에게 다음과 같은 질문을 해보세요:

"다음 정보를 표 형태로 정리해주세요:
- 제품명: 스마트폰 A, 가격: 800,000원, 화면: 6.1인치
- 제품명: 스마트폰 B, 가격: 1,200,000원, 화면: 6.7인치
- 제품명: 스마트폰 C, 가격: 600,000원, 화면: 5.8인치"

예상 결과

AI가 마크다운 테이블로 응답하면, 다음과 같이 표시되어야 합니다:

  • ✅ 완전한 테이블 형태로 표시
  • ✅ 헤더 행은 배경색이 다르게 표시
  • ✅ 각 행이 구분되어 표시
  • ❌ 각 행이 개별적으로 분리되어 표시 (수정 전)

테스트 2: 사이드바 크기 조정

다음 단계를 따라 테스트해보세요:

  1. AI 챗봇 사이드바를 열어주세요
  2. 사이드바 왼쪽 가장자리에 마우스를 올려보세요
  3. 커서가 ↔ 모양으로 변하는지 확인하세요
  4. 마우스를 드래그하여 사이드바 크기를 조정해보세요
  5. 사이드바를 닫았다가 다시 열어서 크기가 유지되는지 확인하세요

예상 결과

  • ✅ 사이드바 왼쪽 가장자리에서 리사이즈 커서 표시
  • ✅ 드래그로 300px ~ 800px 범위에서 크기 조정 가능
  • ✅ 리사이즈 중 시각적 피드백 제공
  • ✅ 조정된 크기가 저장되어 다음에도 유지
  • ✅ 페이지 내용이 사이드바 크기에 맞춰 조정

테스트 3: 복합 기능 테스트

HTML 첨부 + 테이블 요청 + 사이드바 리사이즈를 함께 테스트해보세요:

  1. 사이드바 크기를 원하는 크기로 조정
  2. HTML 첨부 버튼으로 이 페이지 첨부
  3. "이 페이지의 테스트 항목들을 표로 정리해주세요" 질문
  4. 응답된 테이블이 조정된 사이드바 크기에 맞게 표시되는지 확인
